## 一、项目概述

### 1.1 项目定位

SkillHub 是一个**企业级开源 Agent 技能注册中心**，提供技能的发布、发现、安装和管理能力。其核心定位是：

- **技能市场**：团队私有托管的技能市场，支持发布、版本管理、审核
- **自托管部署**：支持在企业防火墙内私有部署
- **CLI 优先**：提供 REST API 和 CLI 工具

### 1.2 现有技术栈

| 层级 | 技术选型 | 版本 | 用途 |
|------|---------|------|------|
| **后端核心框架** | Spring Boot | 3.2.3 | REST API、服务端渲染 |
| **后端语言** | Java | 21 | 业务逻辑 |
| **构建工具** | Maven | 3.x | 依赖管理、构建 |

| **前端语言** | TypeScript | 5.x | 类型安全 |
| **前端构建** | Vite | 5.x | 开发服务器、生产构建 |
| **数据库** | PostgreSQL | 16 | 业务数据存储 |
| **缓存** | Redis | 7 | Session、缓存 |
| **对象存储** | MinIO | - | 技能包文件存储 |

### 2.2 核心表结构

#### 2.2.1 用户与认证

**users**
```sql
CREATE TABLE users (
    id              BIGINT PRIMARY KEY,
    user_id         VARCHAR(64) UNIQUE NOT NULL,  -- 全局唯一标识
    username        VARCHAR(100) UNIQUE NOT NULL,
    display_name    VARCHAR(200),
    email           VARCHAR(255) UNIQUE NOT NULL,
    password_hash   VARCHAR(255),                  -- BCrypt 加密
    bio             TEXT,
    avatar_url      VARCHAR(500),
    created_at      TIMESTAMP NOT NULL,
    updated_at      TIMESTAMP NOT NULL,
    deleted_at      TIMESTAMP,
    enabled         BOOLEAN DEFAULT TRUE
);
```

**user_profiles**
```sql
CREATE TABLE user_profiles (
    id              BIGINT PRIMARY KEY,
    user_id         BIGINT NOT NULL REFERENCES users(id),
    field_name      VARCHAR(100) NOT NULL,         -- 字段名
    value           TEXT,                           -- 字段值
    editable        BOOLEAN DEFAULT TRUE,
    requires_review  BOOLEAN DEFAULT FALSE,
    reviewed_at     TIMESTAMP,
    reviewed_by     BIGINT REFERENCES users(id),
    created_at      TIMESTAMP NOT NULL,
    updated_at      TIMESTAMP NOT NULL
);
```

#### 2.2.2 命名空间

**namespaces**
```sql
CREATE TABLE namespaces (
    id              BIGINT PRIMARY KEY,
    name            VARCHAR(100) UNIQUE NOT NULL,  -- 命名空间名称
    display_name    VARCHAR(200),
    description     TEXT,
    visibility      VARCHAR(20) NOT NULL,          -- PUBLIC/PRIVATE
    owner_id        BIGINT NOT NULL REFERENCES users(id),
    created_at      TIMESTAMP NOT NULL,
    updated_at      TIMESTAMP NOT NULL,
    deleted_at      TIMESTAMP
);
```

**namespace_members**
```sql
CREATE TABLE namespace_members (
    id              BIGINT PRIMARY KEY,
    namespace_id    BIGINT NOT NULL REFERENCES namespaces(id),
    user_id         BIGINT NOT NULL REFERENCES users(id),
    role            VARCHAR(20) NOT NULL,          -- OWNER/ADMIN/MEMBER
    created_at      TIMESTAMP NOT NULL,
    UNIQUE(namespace_id, user_id)
);
```

#### 2.2.3 技能与版本

**skills**
```sql
CREATE TABLE skills (
    id              BIGINT PRIMARY KEY,
    namespace_id    BIGINT NOT NULL REFERENCES namespaces(id),
    slug            VARCHAR(100) NOT NULL,         -- 技能 slug
    name            VARCHAR(200) NOT NULL,
    description     TEXT,
    readme          TEXT,
    icon_url        VARCHAR(500),
    current_version VARCHAR(50),                  -- 当前版本
    state           VARCHAR(20) NOT NULL,          -- DRAFT/PUBLISHED/ARCHIVED
    promoted        BOOLEAN DEFAULT FALSE,         -- 是否推广到全局
    created_by      BIGINT NOT NULL REFERENCES users(id),
    created_at      TIMESTAMP NOT NULL,
    updated_at      TIMESTAMP NOT NULL,
    deleted_at      TIMESTAMP,
    published_at    TIMESTAMP,
    archived_at     TIMESTAMP,
    UNIQUE(namespace_id, slug)
);
```

**skill_versions**
```sql
CREATE TABLE skill_versions (
    id              BIGINT PRIMARY KEY,
    skill_id        BIGINT NOT NULL REFERENCES skills(id),
    version         VARCHAR(50) NOT NULL,          -- 语义版本 1.0.0
    state           VARCHAR(20) NOT NULL,          -- DRAFT/PUBLISHED/DEPRECATED
    source_url      VARCHAR(500),                 -- 包存储路径
    file_size       BIGINT,                       -- 文件大小
    checksum        VARCHAR(64),                   -- SHA-256
    downloads       BIGINT DEFAULT 0,
    published_at    TIMESTAMP,
    created_by      BIGINT NOT NULL REFERENCES users(id),
    created_at      TIMESTAMP NOT NULL,
    UNIQUE(skill_id, version)
);
```

#### 2.2.4 社交与评价

**skill_ratings**
```sql
CREATE TABLE skill_ratings (
    id              BIGINT PRIMARY KEY,
    skill_id        BIGINT NOT NULL REFERENCES skills(id),
    user_id         BIGINT NOT NULL REFERENCES users(id),
    rating          INTEGER NOT NULL CHECK (rating >= 1 AND rating <= 5),
    created_at      TIMESTAMP NOT NULL,
    updated_at      TIMESTAMP NOT NULL,
    UNIQUE(skill_id, user_id)
);
```

**skill_stars**
```sql
CREATE TABLE skill_stars (
    id              BIGINT PRIMARY KEY,
    skill_id        BIGINT NOT NULL REFERENCES skills(id),
    user_id         BIGINT NOT NULL REFERENCES users(id),
    created_at      TIMESTAMP NOT NULL,
    UNIQUE(skill_id, user_id)
);
```

**skill_downloads**
```sql
CREATE TABLE skill_downloads (
    id              BIGINT PRIMARY KEY,
    skill_version_id BIGINT NOT NULL REFERENCES skill_versions(id),
    user_id         BIGINT NOT NULL REFERENCES users(id),
    downloaded_at   TIMESTAMP NOT NULL
);
```

#### 2.2.5 标签系统

**labels**
```sql
CREATE TABLE labels (
    id              BIGINT PRIMARY KEY,
    name            VARCHAR(100) NOT NULL,
    color           VARCHAR(7),                    -- HEX 颜色
    description     TEXT,
    created_at      TIMESTAMP NOT NULL
);
```

**skill_labels**
```sql
CREATE TABLE skill_labels (
    skill_id        BIGINT NOT NULL REFERENCES skills(id),
    label_id        BIGINT NOT NULL REFERENCES labels(id),
    created_at      TIMESTAMP NOT NULL,
    PRIMARY KEY (skill_id, label_id)
);
```

#### 2.2.6 治理与审核

**skill_promotions**（推广申请）
```sql
CREATE TABLE skill_promotions (
    id              BIGINT PRIMARY KEY,
    skill_id        BIGINT NOT NULL REFERENCES skills(id),
    requester_id    BIGINT NOT NULL REFERENCES users(id),
    status          VARCHAR(20) NOT NULL,          -- PENDING/APPROVED/REJECTED
    reviewer_id     BIGINT REFERENCES users(id),
    review_comment  TEXT,
    submitted_at    TIMESTAMP NOT NULL,
    reviewed_at     TIMESTAMP,
    UNIQUE(skill_id)
);
```

**skill_reviews**（审核记录）
```sql
CREATE TABLE skill_reviews (
    id              BIGINT PRIMARY KEY,
    skill_version_id BIGINT NOT NULL REFERENCES skill_versions(id),
    reviewer_id     BIGINT NOT NULL REFERENCES users(id),
    decision        VARCHAR(20) NOT NULL,          -- APPROVED/CHANGES_REQUESTED/REJECTED
    comment         TEXT,
    created_at      TIMESTAMP NOT NULL
);
```

#### 2.2.7 审计日志

**audit_logs**
```sql
CREATE TABLE audit_logs (
    id              BIGINT PRIMARY KEY,
    user_id         BIGINT REFERENCES users(id),
    action          VARCHAR(100) NOT NULL,         -- 动作类型
    entity_type     VARCHAR(50) NOT NULL,          -- 实体类型
    entity_id       BIGINT,                       -- 实体ID
    details         JSONB,                        -- 详细信息
    ip_address      VARCHAR(45),
    created_at      TIMESTAMP NOT NULL
);
```


### 3.3 API 认证机制

**Mock 认证（本地开发）**
```yaml
skillhub:
  auth:
    mock:
      enabled: true  # application-local.yml
```
通过 Header `X-Mock-User-Id: local-user` 模拟用户

**OAuth2 认证（生产）**
- GitHub OAuth2 支持
- 支持 Spring Security OAuth2 Client

**Session Bootstrap（简化登录）**
```yaml
skillhub:
  auth:
    session-bootstrap:
      enabled: true
```
通过一次性码实现无密码登录

### 7.1 功能缺失

```sql
CREATE TABLE employee_profiles (
```
